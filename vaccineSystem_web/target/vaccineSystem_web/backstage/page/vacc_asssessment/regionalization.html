2021年5月4日山西省新型冠状病毒肺炎疫情情况

发布时间： 2021-05-04 来源: 科教科杨蓓 访问量:5 字体： [ A- A+ ]

2021年5月3日0-24时，山西省本地无新增新冠肺炎确诊病例。累计报告确诊病例138例，治愈出院138例。现有疑似病例0例。

2021年5月3日0-24时，山西省无新增境外输入确诊病例。累计报告境外输入性确诊病例113例，治愈出院105例，现有在院隔离治疗病例8例。现有疑似病例0例。

2021年5月3日0-24时，山西省无新增无症状感染者。尚在医学观察的无症状感染者10例。

累计追踪到密切接触者6850人(含境外输入病例密切接触者),当日解除医学观察3人，尚在医学观察的密切接触者26人。
