2021年5月4日山西省疫苗报废情况

发布时间： 2021-05-04 来源: 山西省疾控中心 访问量:10 字体： [ A- A+ ]

疫苗报损制度

为了加强疫苗的安全监管，凡疫苗有霉变、裂开、过期失效等质量不符合要求时，应办理报废手续。

一、公卫科应填写三联报损单，注明药品名称、规格、单位、数量、价格和报损原因，并有报损人、药房管理者和领导签名，特殊情况应上报主管领导审批。

二、疫苗报损采用回收的方式进行，报损疫苗统一交给医疗废弃物回收处再由杭州大地维康公司回收处理。

三、疫苗保管员将报损单装订成册，一联交县疾控中心，一联留底。

过期、失效疫苗处理制度

一、疫苗应严格按计划使用，尽量避免浪费，如有过期报废疫苗严格执行本制度。

二、如本单位发现疫苗存在明显质量问题，应立即停止使用，上报昌平区疾病预防控制中心，申请报废。

三、过期疫苗应妥善保管，不能与其他疫苗混放。

四、要及时清理、上报过期失效疫苗，并做好疫苗处理记录。要核实疫苗数量、批号、处理原因等情况，进行登记、签字，对报废疫苗进行无害化处理后统一销毁。

五、因冷链环节等其它原因导致疫苗报废的应逐级追究责任。
